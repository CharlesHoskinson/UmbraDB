# Test wallets

`preview-test-wallet.json` contains a seed and secret key for a Midnight
**Preview testnet** wallet, generated and funded via the official faucet
(https://faucet.preview.midnight.network/). tDUST has no monetary value and
exists solely to facilitate testnet transactions -- committed here
deliberately so this environment does not need to be re-funded on every
fresh machine. Do not reuse this pattern for mainnet or any key with real
value.
